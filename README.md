# 💫 About Me

Software Engineer with 2+ years building production systems in Node.js and Go — from distributed infrastructure and AI pipelines to zero-knowledge security platforms. Currently volunteering at Pakistan's national cybersecurity agency (PKCERT), shipping internal tooling and public-facing platforms. I build things that are fast, observable, and hard to break.

---

# 🔭 Currently Building

- **[SecureCollab](https://github.com/mubashir05-beep/securecollab)** — Self-hostable, zero-knowledge team messaging platform. Server stores only ciphertext (X25519 + ChaCha20-Poly1305 via Rust/Tauri). Go microservices, CDC pipeline (Debezium → Redpanda → ClickHouse), 1K+ RPS gateway.
- **[AxiomX](https://github.com/mubashir05-beep/AxiomX)** — Institutional-grade order matching engine in Go. Sub-millisecond latency, 1K+ orders/sec, deployed on AWS EKS with full observability stack.
